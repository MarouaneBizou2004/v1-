# MasterMyCity Frontend

React + Vite single-page application for the MasterMyCity chatbot and reporting flow.

## Available Scripts

- `npm run dev` – start the dev server on <http://localhost:5173>.
- `npm run build` – compile TypeScript and bundle assets.
- `npm run preview` – preview the production build.
- `npm run test` – run Vitest unit tests.
- `npm run lint` – lint the source files.

## Features

- Multilingual UI (English, French, Arabic) via i18next.
- Issue report wizard with location, description, and image upload placeholders.
- Chat interface integrated with the backend and Socket.io notifications.
- TailwindCSS styling with RTL support for Arabic.
